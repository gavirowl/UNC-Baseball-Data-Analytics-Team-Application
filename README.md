# UNC-Baseball-Data-Analytics-Team-Application
This project analyzes and attempts to explain Mac Horvath's rise from slump to hot streak. All visualizations and analysis were created using R. The data used in this project come from 2022 Trackman data for UNC baseball.

## Files
- "Horvath2022turnaround.pdf" - Final written report

## Skills Demonstrated
- Visualizing data in R
- Crafting a report and communicating findings
- Data analysis in R
